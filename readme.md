# RecipeHub

Small app to search for something you would like to cook. 

Check out required ingredients depending on the amount of servings needed and link to recipe of the dish you are interested in.

## Getting Started

A few sample queries to check out:

* salad
* pizza
* pasta
* popcorn
* lobster
* steak

More options to look at are [here](https://forkify-api.herokuapp.com/phrases.html)
